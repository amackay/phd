Definition zesq 121
Description: An integer is even iff its square is even. (Contributed by Mario Carneiro, 12-Sep-2015.)
Assertion
Ref Expression
zesq ⊢ (𝑁 ∈ ℤ → ((𝑁 / 2) ∈ ℤ ↔ ((𝑁↑2) / 2) ∈ ℤ))
