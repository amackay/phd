Definition nnind 95
Description: Principle of Mathematical Induction (inference schema). The first four hypotheses give us the substitution instances we need; the last two are the basis and the induction step. See nnaddcl (future) for an example of its use. See nn0ind (future) for induction on nonnegative integers and uzind (future), uzind4 (future) for induction on an arbitrary upper set of integers. See indstr (future) for strong induction. See also nnindALT (future). This is an alternative for Metamath 100 proof #74. (Contributed by NM, 10-Jan-1997.) (Revised by Mario Carneiro, 16-Jun-2013.)
Hypotheses
Ref Expression
nnind.1 ⊢ (𝑥 = 1 → (𝜑 ↔ 𝜓))
nnind.2 ⊢ (𝑥 = 𝑦 → (𝜑 ↔ 𝜒))
nnind.3 ⊢ (𝑥 = (𝑦 + 1) → (𝜑 ↔ 𝜃))
nnind.4 ⊢ (𝑥 = 𝐴 → (𝜑 ↔ 𝜏))
nnind.5 ⊢ 𝜓
nnind.6 ⊢ (𝑦 ∈ ℕ → (𝜒 → 𝜃))
Assertion
Ref Expression
nnind ⊢ (𝐴 ∈ ℕ → 𝜏)
